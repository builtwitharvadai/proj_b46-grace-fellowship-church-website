# Grace Fellowship Church Website

A modern, responsive website for a Nigerian church congregation that provides essential information about services, events, and ministries. The website serves as the primary digital presence to connect with current members and welcome new visitors to the church community.

## Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Event Management**: Dynamic event calendar with upcoming activities
- **Ministry Pages**: Dedicated pages for Children's, Youth, Men's, Women's, and Outreach ministries
- **Contact Forms**: Easy communication with church leadership
- **SEO Optimized**: Enhanced visibility in search engines
- **Progressive Web App**: Offline functionality and fast loading
- **Analytics Integration**: Track visitor engagement and website performance

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Build Tools**: Node.js, PostCSS
- **Analytics**: Google Analytics integration
- **Images**: Unsplash API integration
- **Deployment**: Static site hosting
