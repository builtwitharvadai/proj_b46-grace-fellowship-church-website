# Getting Started with Grace Fellowship Church Website

Welcome to the Grace Fellowship Church website content management system! This guide will help you get started with managing and updating website content, even if you don't have technical experience.

## What You'll Learn

This guide covers:
- Understanding the website system
- Your first steps as a content manager
- Basic concepts and terminology
- Where to find help
- Essential tasks you'll perform regularly

## Understanding the System

### What is the Website Built With?

The Grace Fellowship Church website is a modern, static website that stores all content in simple text files called JSON files. Think of JSON files as organized lists of information that the website reads and displays.

**Key Benefits:**
- **Simple:** Content is stored in easy-to-read text files
- **Safe:** Changes are tracked and can be undone
- **Fast:** Website loads quickly for visitors
- **Reliable:** No complex database to maintain
